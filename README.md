## 🖼️ Adding Product Images

Place product images in `/assets/images/`:
- `og-image.jpg` (1200×630px) — Social media share image
- `logo.png` (200×200px) — Business logo
- `product-pulsar-tank.jpg` — Pulsar fuel tank
- `product-led-bulb.jpg` — LED headlight
- etc.

Then replace `product-placeholder` divs in HTML with `<img>` tags:
```html
<img src="assets/images/product-name.jpg" alt="Product Name" loading="lazy" />
```
